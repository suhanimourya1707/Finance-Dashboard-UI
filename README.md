

# FinFlow — Finance Dashboard UI

A modern and interactive Finance Dashboard UI built using HTML, CSS, and JavaScript. This project helps users track income, expenses, and overall financial activity with clear visualizations and an intuitive interface.

---

## Features

* Interactive charts using Chart.js
* Overview of total balance, income, and expenses
* Transaction management (add, view, filter, search)
* Advanced filtering and sorting options
* Monthly balance trends and spending insights
* Dark and Light mode toggle
* Role-based UI (Admin and Viewer)
* Export data as CSV and JSON
* Fully responsive design

---

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Charts: Chart.js
* Storage: LocalStorage

---

## Project Structure

```
├── index.html     # Main UI structure
├── style.css      # Styling and themes
├── app.js         # Logic and functionality
```

---

## How It Works

* Uses localStorage to store transaction data
* Displays financial summaries dynamically
* Generates charts for trends and category breakdown
* Supports real-time updates when transactions change
